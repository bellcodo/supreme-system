# supreme-system

1. Create a heroku account.
2. Create a git repo.
3. Add `requirements.txt`
4. Add `Procfile`
5. Add `hello_app.py`
6. Create a new heroku app using GitHub option.
7. Search for `your-repo-name` and connect to it.
8. Scroll to the bottom and run `manual deploy` master.
9. View the app.
10. Troubleshoot!
